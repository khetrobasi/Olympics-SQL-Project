Olympics SQL Project
======================
Project Overview
===================
	This project analyzes 120 years of Olympic Games data using SQL Server. The dataset contains athlete information, Olympic events, countries, sports, medals, and participation history.

	The objective of this project is to demonstrate SQL skills including data import, data cleaning, joins, aggregate functions, window functions, CTEs, subqueries, and business insights.

Dataset
========

The project uses the Kaggle dataset:
====================================
- athlete_events.csv
- noc_regions.csv

Total Records:
==============
 Athlete Events: 271,116
 NOC Regions: 230+

Tools Used
============
- SQL Server 2022
- SQL Server Management Studio (SSMS)
- GitHub

Database Objects
===================
Tables : athlete_events & noc_regions

SQL Concepts Covered
====================
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- INNER JOIN
- LEFT JOIN
- CTE
- Window Functions
- Subqueries
- CASE Statement
- Views

📈 Business Questions

- Which country won the most Gold medals?
- Which sport has the highest participation?
- Which athletes won the most medals?
- Gender participation analysis.
- Medal trend over the years.
- Top Olympic countries.
